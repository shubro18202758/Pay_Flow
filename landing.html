Architecture

Seven-Layer Intelligence Pipeline

Every transaction passes through all seven layers in under 100ms — from interception to verdict.

Layer 1
Real-Time Transaction Interception
WebSocket gateway intercepts every UPI/NEFT/RTGS/IMPS transaction before approval. Zero-latency streaming ingestion with CRC32 checksums and structural validation.
FastAPI WebSocket msgspec AsyncIO
Layer 2
42-Dimensional Feature Engineering
Velocity tracking (5-min / 1-hr / 24-hr windows), behavioral Z-score deviation, text anomaly detection with homoglyph and entropy analysis. Produces (N, 30) vectorized float32 arrays.
VelocityTracker BehavioralAnalyzer TextAnomalyAnalyzer
Layer 3
ML Fraud Detection Ensemble
XGBoost (known patterns, AUC > 0.95) + Isolation Forest (novel anomalies) + PyTorch Autoencoder (latent reconstruction error). GPU-accelerated with automatic CPU fallback. SMOTE-balanced training.
XGBoost IsolationForest Autoencoder SMOTE CUDA
Layer 4
Graph Intelligence Engine
Dynamic directed multigraph construction with money mule star-topology detection, DFS-based cycle detection, and k-hop subgraph extraction for GNN scoring. 278K-parameter FraudGAT with 3-layer Graph Attention Networks.
NetworkX PyG FraudGAT GATConv
Layer 5
Risk Scoring & Alert Triage
EMA-based adaptive threshold with 3-tier classification (HIGH → Graph + LLM, MEDIUM → Graph, LOW → drop). Bounded async dispatch with 10K queue backpressure and max-2-retry policy.
DynamicThreshold AlertRouter 10K Queue
Layer 6
LLM Investigation Agent
Qwen 3 via Ollama with Chain-of-Thought reasoning. Stateful investigator agent with tool-calling for graph queries, ML feature analysis, and blockchain log reads. Human-in-the-Loop escalation below confidence threshold.
Qwen 3 Ollama CoT Reasoning HITL
Layer 7
Tamper-Evident Audit Ledger & Regulatory Reporting
SHA-256 hash-chain with Ed25519 signatures, Merkle tree checkpoints, and ZKP-verified compliance proofs. Circuit breaker smart contracts for autonomous account freezing. Auto-generated STR / FMR / CTR PDFs for FIU-IND.
SHA-256 Ed25519 ZKP MerkleTree ReportLab
Capabilities

End-to-End Intelligence
at Every Layer

From transaction interception to regulatory filing — every capability is production-ready and fully autonomous.

🕸️

Graph-Based Mule Detection

Star-topology analysis identifies money mule convergence patterns — the high-fan-in accounts that funnel proceeds from dozens of victims into single extraction points.

🔁

Cycle Detection Engine

DFS-based circular flow detection with configurable max depth. Catches round-tripping laundering patterns (A→B→C→A) that rule-based systems completely miss.

🧠

LLM Forensic Investigator

Qwen 3 reasons step-by-step through flagged cases using Chain-of-Thought, querying the graph and ML layers as tools to produce full forensic narratives.

🔒

Blockchain Audit Trail

Every AI decision is anchored in a SHA-256 hash-chain with Ed25519 signatures. Tamper-evident, restart-resilient, and averages 0.365ms per block.

⚡

Smart Contract Circuit Breaker

Autonomous account freezing when multi-model consensus reaches critical threshold. Millisecond execution with cryptographic proof logged for compliance.

🛡️

Zero-Knowledge Compliance

ZKP verification validates KYC/CFR status without exposing PII. Privacy-preserving by design — compliant with RBI data protection guidelines.

📊

Behavioral Deviation Scoring

Per-account rolling 90-day behavioral profiles with Z-score deviation detection across amounts, frequencies, geolocations, and device fingerprints.

🔤

Text Anomaly Analysis

Detects Cyrillic homoglyph attacks, high-entropy strings, and social engineering patterns in beneficiary names and transaction notes - preventing SWIFT-style spoofing.
